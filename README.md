A chill, cute focus timer featuring a relaxed Capybara in a hot spring.

- **Themes**: Day (Teal), Sunset (Purple/Orange), Night (Dark Blue).
- **Categories & Presets**:
  - **Kitchen**: Ramen (3m), Soft Egg (6m), Jammy Egg (7m), Hard Egg (10m).
  - **Productivity**: Pomodoro (25m), Breaks (5m/15m).
  - **Wellness**: Power Nap (20m), Deep Breath (1m).
- **Sinking Mechanics**: The Capybara slowly sinks deeper as the timer progresses!
- **Alarm**: A gentle beep sounds when the timer is done.

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000).
